Glassdoor Reviews NLP


By analyzing Glassdoor reviews, I seek to uncover how experiences and perceptions differ between various positions at the same company, as well as same job titles in different companies. Using the KeyBERT model in Python, I extract keywords using both Maxsum and MMR (Maximal Marginal Relevance) methods.
